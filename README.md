# Smart Calculator
A web-based calculator built with pure JavaScript, HTML, and CSS as a learning project.

## Features
- Basic arithmetic: addition, subtraction, multiplication, and division
- Percentage calculation
- Keyboard support

## Built With
- JavaScript  
- HTML  
- CSS  

## How to Run
1. Open the live demo link.
2. Start using the calculator in your browser.
